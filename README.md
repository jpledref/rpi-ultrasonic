# Raspberry Pi Ultrasonic

This application uses the ultrasonic sensor to calculate a distance.

Trigger pin: GPIO #05 (BCM)
Echo pin: GPIO #06 (BCM)
VCC pin: VCC pin 
GND pin: GND pin
